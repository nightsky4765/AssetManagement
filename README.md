# AssetManagement APP

A modern, mobile-first Web Application for managing personal finances alongside tailored asset allocation advice.

## Features

- 💸 **Income & Expense Tracking**: Log daily transactions seamlessly. Cash balances immediately integrate with your overall portfolio.
- 🎯 **Smart Asset Allocation**: Decide your custom split for emergency funds, stocks, and bonds. The bi-directional algorithm will analyze your actual assets against your target strategy.
- 📊 **Rich Analytics**: Multi-perspective charting including expense/income pie charts and bar charts to digest where your cash flows.
- 🌙 **Premium Dark UI**: Extremely modern aesthetic, completely responsive and perfect for deployment as a PWA or Native App via Capacitor.
- 🌍 **Internationalization**: Full dual-language support (English / Traditional Chinese).

## Quick Start (For Developers)

\`\`\`bash
# Install dependencies
npm install

# Start Local Dev Server
npm run dev
\`\`\`

## 繁體中文說明

這款極簡風格的手機優先 (Mobile-first) Web App 專為個人理財所設計，結合了基礎收支記帳與客製化的資產配置建議。

## 核心亮點

- 💸 **即時收支記帳**：順暢輸入日常花費，現金流餘額會與總資產池即時連動。
- 🎯 **聰明資產配置演算法**：完全客製化您的緊急預備金與生活開銷額度，剩下的資金由系統根據您的戰略比例進行 股票/債券 的雙向再平衡分析。
- 📊 **多維度統計報表**：內建業界標準的 Chart.js 視覺化圖表，用圓餅圖、長條圖輕鬆掌握流向。
- 🌙 **高質感深色主題**：極具未來感的深色 UI 設計，介面不凌亂，專注於您的資產成長。
- 🌍 **完整雙語系**：支援英文與繁體中文，自由一鍵切換。
